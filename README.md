# 🎬 CineVerse AI

CineVerse AI is a movie recommendation platform that combines content-based filtering with live TMDB data to help users discover movies tailored to their interests.

The application provides real-time search, detailed movie information, trending titles, and personalized recommendations through an interactive web interface built with Streamlit and FastAPI.

## Live Demo

https://supremeinferno-movies-recommender.streamlit.app

## Features

- Intelligent movie recommendations using TF-IDF similarity
- Genre-based recommendation engine
- Real-time movie search
- Trending, Top Rated, Upcoming, and Now Playing feeds
- Detailed movie information pages
- TMDB API integration
- Modern responsive user interface
- FastAPI-powered backend services

## Recommendation Approach

### Content-Based Filtering
Movies are represented using TF-IDF vectors generated from metadata and textual features. Cosine similarity is used to identify and recommend movies with similar characteristics.

### Genre-Based Recommendations
Additional recommendations are generated by analyzing genre overlap to improve recommendation diversity and discovery.
